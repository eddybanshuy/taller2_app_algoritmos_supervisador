# taller2_app_algoritmos_supervisador
Taller 2: Aplicación de algoritmos supervisados



Para la siguiente actividad se deberá tomar el dataset cargado en el CANVAS y, con base en estos datos, presentar lo siguiente:

Aplicar algoritmo árbol de decisión, utilice RandomizedSearchCV.
Aplicar algoritmo RandomForest, utilice RandomizedSearchCV.
Aplicar algoritmo KNN, utilice RandomizedSearchCV.
Aplicar algoritmo SVM, utilice RandomizedSearchCV.
Cada algoritmo debe tener su matriz de confusión y el reporte de las métricas precisión, recall, accuracy y F1-Score.
